# 📝 Todo App (React)

Простое приложение для управления списком задач, созданное на React ⚛️

## ✨ О проекте

Todo App — это минималистичное приложение для создания списка дел.  
Позволяет удобно добавлять, отмечать и удалять задачи, сохраняя их состояние в браузере 💾

## 🛠️ Технологии

- ⚛️ React — логика и UI  
- ⚡ Vite — сборка и запуск проекта  
- 🎨 SCSS — стилизация интерфейса  

## ⚙️ Функционал

- ➕ Добавление новых задач  
- ✔️ Отметка задач как выполненных  
- 🗑️ Удаление задач  
- 💾 Сохранение данных в localStorage  
- 🔄 Восстановление списка после перезагрузки страницы
